Modeling Humans in VR

This project explored the use of Virtual Reality for recording and interpreting human activities. Using Unreal Engine as the basis, I implemented a framework for capturing multi-agent interactions and parsing them into a formal action grammar. The system was able to match observed contact events with modeled actions in real-time, enabling structured datasets for robotics and activity understanding.

The recorded sessions were also used to generate synthetic datasets, later stored and made available via the openEASE knowledge service. This provided a valuable bridge between immersive VR experiences and symbolic reasoning systems in robotics.

Project highlights

  • Recording of human motion and object interaction in VR
  • Parsing activities into a formal action grammar
  • Support for multi-agent interaction modeling
  • Integration with openEASE for dataset storage and reuse
  • Applications in activity recognition and synthetic data generation

Role: I designed and implemented the VR recording framework, extended Unreal Engine functionality for structured data capture, and supervised a student project extending the system for multi-agent interactions. I also authored publications presenting the approach at leading venues in robotics (ICRA 2018) and VR research.
